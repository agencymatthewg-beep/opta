? What is Netlify?

Netlify is a cloud platform for hosting web applications. Think of it as a specialized web server that:

  • Hosts your app 24/7 - Your app is always online, accessible from anywhere
  • Handles traffic spikes - Automatically scales when many people visit
  • Provides free SSL - Secure HTTPS connection included
  • Global CDN - Fast loading worldwide via edge servers
  • Continuous deployment - Push code, it deploys automatically

Why Do You Need It?

To make lm.optamize.biz work, your Life Manager app needs to be running on a server somewhere on the internet. Netlify provides that server (for free) and connects it to your custom domain.

+ Alternative: Vercel via GitHub

If you prefer Vercel (better for Next.js long-term), you can bypass the CLI rate limit by connecting your GitHub repository directly in the Vercel dashboard:

  1. Go to vercel.com/new
  2. Import your GitHub repository (agencymatthewg-beep/opta)
  3. Set root directory to opta-life-manager
  4. Deploy and add lm.optamize.biz as custom domain
