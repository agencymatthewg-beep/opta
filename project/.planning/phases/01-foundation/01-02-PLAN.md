<task type="auto">
  <name>Task 2: Create cn utility helper</name>
  <files>apps/web/AICompare/src/lib/utils.ts</files>
  <action>
  Create the standard cn (classNames) helper used throughout components:

  ```typescript
  import { type ClassValue, clsx } from "clsx";
  import { twMerge } from "tailwind-merge";

  export function cn(...inputs: ClassValue[]) {
    return twMerge(clsx(inputs));
  }
  ```

  Also install clsx: `cd apps/web/AICompare && npm install clsx`

  This utility:
  - Merges Tailwind classes intelligently (twMerge)
  - Handles conditional classes (clsx)
  - Standard pattern used with CVA components

  AVOID: Creating multiple utility files. Keep this as the single entry point for class utilities.
  </action>
