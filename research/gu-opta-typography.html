The Official Font Specification

Primary Font: Sora

Sora is a geometric sans-serif with excellent legibility and a modern, technical feel. It conveys precision and sophistication without being cold.

/* Google Fonts Import */ @import url('https://fonts.googleapis.com/css2?family=Sora:wght@300;400;500;600;700&display=swap'); /* CSS Variable */ --font-primary: 'Sora', -apple-system, BlinkMacSystemFont, sans-serif;

Hero Heading Style

The signature Opta heading style — used for page titles, brand moments, and major visual statements.

Property Value Notes
Font Family Sora With system fallbacks
Font Size 3.5rem (56px) Scale down for smaller contexts
Font Weight 700 (Bold) Always bold for hero
Letter Spacing 0.12em Wide but not extreme
Line Height 1.1 Tight for display text
Fill Moonlight Gradient White → Violet → Indigo
Text Shadow 0 0 80px rgba(139, 92, 246, 0.5) Purple atmospheric glow
/* CSS Implementation */ .opta-hero-heading { font-family: 'Sora', sans-serif; font-size: 3.5rem; font-weight: 700; letter-spacing: 0.12em; line-height: 1.1; /* Moonlight Gradient */ background: linear-gradient(180deg, #fafafa 0%, #a855f7 50%, #6366f1 100%); -webkit-background-clip: text; -webkit-text-fill-color: transparent; /* Glow */ text-shadow: 0 0 80px rgba(139, 92, 246, 0.5); }
