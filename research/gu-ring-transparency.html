CRITICAL DESIGN RULE

THE FUNDAMENTAL PRINCIPLE

The Opta Ring's center hole must remain completely transparent. All visual elements ‚Äî obsidian glass, plasma glow, reflections ‚Äî are contained within the torus wall only. The ring floats as a 3D object, revealing whatever background lies beneath it.

Background Independence

The ring must work on ANY background ‚Äî the center is always a window through to the content behind.

The Ring as a 3D PNG

Think of the Opta Ring like a donut-shaped PNG with transparency. The center hole is literally empty ‚Äî it's a window through to whatever lies behind the ring.

This allows the ring to be placed on top of any content ‚Äî dashboards, images, gradients, videos ‚Äî while maintaining its identity as a 3D floating object.

  • The torus WALL contains all visual effects
  • The center HOLE is always transparent
  • No radial gradients that bleed into center
  • No ambient glow that fills the hole
  • The ring can move freely over any surface

Visual Comparison

WRONG: Filled Center

The center has a gradient fill. The ring cannot be placed over content ‚Äî it obscures what's behind it.

CORRECT: Transparent Center

The center is fully transparent. The checkered pattern (representing any background) shows through perfectly.

Implementation Rules

DO: Transparent Background

Set the ring's center to background: transparent or background: none. The torus is a hollow tube.

DO: Clip Glow to Wall

Any glow effects (plasma, ambient) must be clipped or masked to stay within the torus wall boundaries.

DO: Use Border for Torus

In CSS, use a thick border to create the torus wall, not a filled circle with a mask.

DON'T: Radial Gradients Through Center

Never use radial-gradient() that extends into the center of the ring ‚Äî this creates a visible fill.

DON'T: Inset Box Shadows That Fill

Avoid large inset box-shadow values that bleed into the center. Keep them tight to the wall.

DON'T: Ambient Glow Behind Ring

The ambient purple fog should be BEHIND the ring in z-order, not filling the ring's center.

Code Examples

SwiftUI Implementation

// CORRECT: Ring as a stroke (hollow center) Circle() .strokeBorder( LinearGradient(colors: [.obsidianLight, .black], ...), lineWidth: 50 ) // The Circle() has NO .fill() - center is transparent // CORRECT: Plasma contained in stroke area Circle() .strokeBorder(plasmaGradient, lineWidth: 4) .padding(25) // Inset to stay within torus wall

CSS Implementation

/* CORRECT: Border creates the torus wall */ .opta-ring { width: 300px; height: 300px; border-radius: 50%; border: 50px solid #1a1a20; background: transparent; /* KEY: No fill! */ } /* CORRECT: Glow only on the border, not filling center */ .opta-ring { box-shadow: 0 0 60px rgba(88, 28, 135, 0.5), /* Outer glow */ inset 0 0 20px rgba(168, 85, 247, 0.15); /* Subtle inner */ }

What NOT To Do

/* WRONG: This fills the center! */ .opta-ring { background: radial-gradient(circle, #09090b, transparent); /* The gradient extends into the center */ } /* WRONG: Large inset shadow bleeds into center */ .opta-ring { box-shadow: inset 0 0 80px rgba(168, 85, 247, 0.4); /* This creates a visible purple wash in the center */ }
