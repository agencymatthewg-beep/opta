Impact: Medium • Security • Manual action required

46 files with explicit `any` type

Impact: Medium • 1-Apps/1C-MacOS/1C2-Optamize-MacOS/src/

Python subprocess bottleneck in Rust backend

Every Tauri command spawns a Python process (30s timeout). No caching, no persistent process. 100-500ms latency per call.
