# Getting Started with Opta

This guide walks you through your first optimization session with Opta.

## Installation

### System Requirements
- **Windows**: Windows 10 (1903+) or Windows 11
- **macOS**: macOS 10.13+ (Apple Silicon or Intel)
- **Linux**: Ubuntu 20.04+ / Debian 11+ / Fedora 35+
- **RAM**: 8GB minimum (for local AI)
- **Disk**: 500MB free space

### Download

1. Go to [Releases](https://github.com/yourusername/opta/releases)
2. Download the installer for your platform
3. Run the installer and follow prompts

## First Launch

When you first open Opta, you'll be greeted with a quick onboarding:

### Question 1: What's your priority?
Choose what matters most to you:
- **Maximum FPS** - Prioritize frame rate above all
- **Visual Quality** - Keep games looking great
- **Balanced** - Best of both worlds

### Question 2: How technical are you?
This sets your experience level:
- **Keep it simple** - Plain language, safer optimizations
- **Standard** - Good balance of power and simplicity
- **Show me everything** - Full control, all options visible

### Question 3: What games do you play?
This helps Opta prioritize optimizations:
- Competitive (Valorant, CS2, Apex)
- Story/Adventure (Elden Ring, Cyberpunk)
- Both

## The Dashboard

After onboarding, you'll see the main dashboard:

### Hardware Meters
Three circular meters show real-time:
- **CPU Usage** - Current processor load
- **GPU Usage** - Graphics card load
- **Memory Usage** - RAM consumption

### Process List
Shows running processes that might impact gaming:
- Category (System, App, Background)
- CPU and memory usage
- Click to see details or terminate

### Quick Actions
One-click optimizations:
- **Stealth Mode** - Kill unnecessary background processes
- **Optimize Game** - Apply settings to detected games

## Your First Optimization

1. Navigate to **Games** in the sidebar
2. Opta auto-scans for installed games
3. Click a game to see optimization options
4. Review suggested changes (Opta explains each one)
5. Click **Apply** to optimize
6. Check your **Score** to see improvement

## Understanding Your Score

The Opta Score has three dimensions:

### Performance (40%)
- FPS Potential
- Stability (no stutters)
- Load Times

### Experience (35%)
- Visual Quality
- Thermal Efficiency
- System Responsiveness

### Competitive (25%)
- Input Lag
- Network Latency
- Background Interference
