# OptaPlus

## Quick Context
- Opta application source code and documentation
- Contains: iOS, macOS, web, CLI implementations
- Use for: building and extending Opta products


Cross-platform iOS/macOS productivity and life management app. Built with Swift for native performance.

## Contents
- **Shared/** - Shared Swift code and models used by iOS and macOS targets
- **iOS/** - iOS app (OptaPlusIOS)
- **macOS/** - macOS app (OptaPlusMacOS)
- **ideas/** - Feature proposals and research

## Usage
Open .xcodeproj in Xcode and select iOS or macOS target. Build with ⌘B or run with ⌘R.

Requires:
- Xcode 15+
- iOS 17+ / macOS 14+ deployment targets
- Swift 5.9+
