# Keyboard Shortcuts

## Navigation

| Shortcut | macOS | iOS | Description |
|----------|-------|-----|-------------|
| `⌘1`–`⌘6` | ✅ | — | Switch to bot 1–6 |
| `⌘P` | ✅ | — | Open command palette |
| `⌘F` | ✅ | — | Search messages |
| `Esc` | ✅ | — | Close panel / dismiss overlay |

## Chat

| Shortcut | macOS | iOS | Description |
|----------|-------|-----|-------------|
| `⏎` (Return) | ✅ | ✅ | Send message |
| `⇧⏎` (Shift+Return) | ✅ | ✅ | Insert new line |
| `⌘K` | ✅ | — | Clear chat |
| `Esc` | ✅ | — | Abort generation |

## Window

| Shortcut | macOS | iOS | Description |
|----------|-------|-----|-------------|
| `⌘N` | ✅ | — | New window |
| `⌘,` | ✅ | — | Open settings |
| `⌘W` | ✅ | — | Close window |
| `⌘/` | ✅ | — | Show keyboard shortcuts cheat sheet |

## Sessions

| Shortcut | macOS | iOS | Description |
|----------|-------|-----|-------------|
| `⌘]` | ✅ | — | Toggle session drawer |
| `⌘P` → "New Session" | ✅ | — | Create new session via command palette |

## iOS Gestures

| Gesture | Description |
|---------|-------------|
| Pull down | Refresh / load newer messages |
| Long press message | Copy message text |
| Tap code block | Copy code to clipboard |
| Swipe back | Navigate to bot list |
