# opta-cli-ts

## Quick Context
- Opta application source code and documentation
- Contains: iOS, macOS, web, CLI implementations
- Use for: building and extending Opta products


Modern TypeScript implementation of the Opta CLI. Enhanced AI coding assistant with multi-provider support and advanced features.

## Contents
- **src/** - TypeScript source code (commands, core, memory, providers, skills, UI)
- **tests/** - Test suite
- **docs/** - Documentation and research
- **scripts/** - Build and utility scripts

## Usage
Install and run:
```bash
npm install
npm run dev          # Development with hot reload
npm run build        # Build for production
npm start            # Run production build
npm test             # Run tests
```

Features:
- Multiple LLM provider support (Anthropic, OpenAI, Google, Groq, etc.)
- MCP (Model Context Protocol) integration
- Memory and persistence layer
- Advanced CLI UI
- Skill system for extensibility

## Quick Start
```bash
npm run dev
```

See docs/ for detailed documentation.
