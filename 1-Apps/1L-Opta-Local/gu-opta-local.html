03 · Setup
Getting Started

Three steps to chat with your local AI from the browser.

1

Start the LMX Server

On your Mac Studio, ensure Opta LMX is running at 192.168.188.11:8000. It serves the OpenAI-compatible API and admin dashboard endpoints.

2

Launch the Web App

cd 1-Apps/1L-Opta-Local/web && npm run dev — Opens at localhost:3004. Enter your Mac Studio's IP and admin key in Settings.

3

Start Chatting

Navigate to Chat, pick a loaded model from the dropdown, and start a conversation. Streaming tokens appear in real-time with the Opta glass UI.

★ WAN access: For remote access, configure a Cloudflare Tunnel in Settings → Tunnel. Named tunnels are required (quick tunnels don't support SSE). The app auto-detects LAN vs WAN and failovers seamlessly.

LMX API Endpoints

The 9 endpoints that Opta Local talks to. All served by your Mac Studio.

EndpointMethodPurpose
/v1/chat/completionsPOSTStreaming chat with loaded models
/v1/modelsGETList all available models
/admin/statusGETServer health snapshot (VRAM, temp, uptime)
/admin/eventsSSELive metrics stream for dashboard
/admin/models/loadPOSTLoad a model into VRAM
/admin/models/unloadPOSTUnload a model from VRAM
/admin/models/{id}/infoGETModel details + VRAM estimate
/admin/sessionsGETList CLI sessions on Mac Studio
/admin/sessions/{id}GETGet session messages for resume
Authentication
Admin endpoints: X-Admin-Key header. Chat API: Bearer token or no auth (configurable). Keys encrypted in storage.
Streaming Protocol
Chat: ReadableStream via fetch. Dashboard: EventSource SSE with auto-reconnect via @microsoft/fetch-event-source.

11 · Decisions
Key Design Decisions

The trade-offs and choices that shaped the product.

Web First, iOS Second

Web iterates faster and validates API patterns that iOS inherits. Browser-to-LMX communication patterns were proven in weeks, not months.

Direct Browser-to-LMX

No intermediate backend. The browser talks directly to the Mac Studio via HTTP/SSE. Simpler architecture, lower latency, one less service to maintain.

Named Tunnels Only

Quick Cloudflare tunnels don't support SSE. Named tunnels are mandatory for the dashboard's real-time metrics stream to work over WAN.

Encrypted Credentials

Admin keys in localStorage are AES-GCM encrypted via Web Crypto API with PBKDF2 key derivation. iOS uses native Keychain Services.
