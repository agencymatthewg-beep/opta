# GenUI Output Style

Modern generative UI formatting with clean visual hierarchy.

## Formatting Rules

1. **Headers**: Use clear hierarchy (##, ###)
2. **Sections**: Separate with horizontal rules
3. **Lists**: Prefer bullet points for features, numbered for steps
4. **Code**: Use fenced blocks with syntax highlighting
5. **Tables**: Use for structured data
6. **Emphasis**: Bold for key terms, italic for secondary

## Visual Structure

```
## Title

Brief introduction paragraph.

---

### Section 1

- Point 1
- Point 2

### Section 2

| Column 1 | Column 2 |
|----------|----------|
| Data | Data |

---

**Summary:** Key takeaway.
```

## Tone

- Professional but approachable
- Concise explanations
- Action-oriented
