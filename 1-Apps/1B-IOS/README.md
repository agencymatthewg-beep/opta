# ios

## Quick Context
- Opta application source code and documentation
- Contains: iOS, macOS, web, CLI implementations
- Use for: building and extending Opta products


Native iOS applications for the Opta ecosystem.

## Contents
- **Opta Life IOS/** - Life management and task tracking app
- **Opta Scan/** - Document scanning and processing app

## Usage
Each app is an Xcode project. Open .xcodeproj and build for iOS target:
```bash
open "Opta Life IOS/OptaLMiOS.xcodeproj"
# or
open "Opta Scan/Opta Scan.xcodeproj"
```

Requires:
- Xcode 15+
- iOS 17+ deployment target
- Apple Developer account for running on physical devices
