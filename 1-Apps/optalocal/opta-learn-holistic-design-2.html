What is LMX?

Opta LMX is the high-performance local inference engine at the heart of the Opta Local ecosystem. It is designed to abstract away the complexity of managing ML accelerators, memory allocation, and model formatting, providing a unified API for running GGUF models seamlessly.

Why it Works

Hardware Introspection

LMX reads your system specs at startup, configuring offload layers automatically.

Unified KV Cache

Optimized memory management prevents context-window blowouts during long chats.

The LMX Daemon

Unlike typical CLI tools, LMX runs as a background daemon. This allows your models to stay loaded in VRAM, making subsequent prompts instant instead of waiting for a 10GB model to load from disk every time.
