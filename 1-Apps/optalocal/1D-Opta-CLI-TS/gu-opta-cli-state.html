‚ö† Headless Session Blocker
Even at Level 5 CEO mode, when pauseAt is reached the agent blocks waiting for user input ‚Äî breaking any unattended run. The fix is an autonomous circuit breaker strategy: when no TTY is present or autonomy.headlessContinue: true is set, checkpoint state to the learning ledger, emit a progress summary, and automatically continue rather than blocking.
