# Session Format

Session logs are written on session close in YJS-compatible markdown.

## Filename

`YYYY-MM-DD-HHMM-{device}-{summary}.md`

## Required Frontmatter

- `date`
- `time`
- `device`
- `user`
- `model`
- `duration`

## Required Sections

- `Summary`
- `Files Changed`
- `Status Changes`
- `Decisions Made`
- `Issues Encountered`
- `Next Steps`
- `Notes`
