# Opta Accounts Features

Opta Accounts (`1R-Opta-Accounts`) is the central identity, SSO, and capability control system for the Opta Local ecosystem.

## Authentication & Identity

- [x] Next.js App Router — foundation with Supabase integration
- [x] Magic link authentication — passwordless email sign-in
- [x] OAuth providers — Google sign-in integration
- [x] Session management — Supabase cookie-based sessions
- [x] Profile management — user settings and account details

## Ecosystem Integration

- [x] SSO portal — centralized login for optalocal.com and other apps
- [x] Capability model — role-based access to Opta features
- [x] Supabase backend — robust user data and schema management
- [ ] Subscription management — Stripe integration for Opta Plus

## Aesthetic & UI

- [x] Opta design system — consistent typography (Sora) and colors
- [x] Glassmorphism UI — elevated aesthetic and modern look
- [x] Responsive design — mobile-friendly authentication flows
