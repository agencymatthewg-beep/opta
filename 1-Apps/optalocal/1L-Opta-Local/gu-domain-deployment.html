‚ö†Ô∏è Mixed Content: The One Gotcha

When served over HTTPS, browsers permanently block HTTP requests to LAN endpoints (192.168.188.11). This means LAN direct mode is unavailable from the deployed domain ‚Äî all traffic routes through the Cloudflare Tunnel instead. Adds ~20-40ms latency vs LAN direct.

Already handled: connection.ts line 160-162 checks window.location.protocol and skips LAN probing on HTTPS. Zero code changes needed.

One-Time Migration Note

The encrypted admin key in localStorage uses PBKDF2 with window.location.origin as the salt. When you switch from localhost:3004 to local.optamize.biz, the origin changes and the old encrypted key won't decrypt. You'll need to re-enter the admin key once on the new domain. This is a one-time step ‚Äî after that, the key persists in the new origin's localStorage.
