Shared Design Rules — All Studios
Shell
.opta-studio-shell + .opta-studio-shell--embedded. Same glassmorphism, same border radius, same grid dot pattern.
Entry/Exit Animation
settingsLayerVariants (Framer Motion). Same scale + blur + translate depth illusion. mode="wait" in AnimatePresence.
Logo System
Press Start 2P, 7-layer pixel shadow stack per-accent. .opta-studio-logo-reserve with is-docked (always). is-active in fullscreen only.
Fullscreen
Shared studioFullscreen state in App. Shift+Space toggles. .v1-settings-overlay-expanded + .opta-studio-shell--fullscreen on open.
Mutual Exclusivity
Only one overlay at a time. Opening a Studio auto-closes Settings. Ctrl+S auto-clears activeStudio. isAnyOverlayActive = settings OR studio.
Lazy Loading
All three Studios in lazyAppModules.tsx as React.lazy(). Zero upfront JS cost. No Suspense fallback needed (instant load).
State Architecture in App.tsx
// Existing Settings layer machine
settingsLayer: 1 | 2 | 3   ← Ctrl+S, L2 grid, L3 deep
settingsFullscreen: boolean

// New Feature Studio state
activeStudio: 'browser' | 'models' | 'atpo' | null
studioFullscreen: boolean

// Combined overlay flag
isAnyOverlayActive = settingsLayer > 1 || activeStudio !== null

// Mutual exclusivity rule
Opening a Studio: if (settingsLayer > 1) goToSettingsLayer(1)
Opening Settings: setActiveStudio(null) [implicit via Ctrl+S]

// Same render container — AnimatePresence mode="wait"
{settingsLayer === 2 && <LazySettingsView />}
{settingsLayer === 3 && <LazySettingsModal />}
{settingsLayer === 1 && activeStudio === 'browser' && <LazyBrowserStudio />}
{settingsLayer === 1 && activeStudio === 'models' && <LazyModelsStudio />}
{settingsLayer === 1 && activeStudio === 'atpo' && <LazyAtpoStudio />}
