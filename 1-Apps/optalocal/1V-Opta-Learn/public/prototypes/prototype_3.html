Identity Management Protocol

Reference-first guide for managing Opta authentication.

> 1. Overview

Opta Accounts provides identity and authentication across the ecosystem, utilizing Identity Blue as its primary accent. This maintains trust-first layouts and structural readability.

> 2. Web Token Structure

Below defines the standardized structure for JWT exchange inside the local dashboard bridging context. 

curl -X POST https://accounts.optalocal.com/v1/auth \
  -H "Content-Type: application/json" \
  -d '{"client_id": "local_dashboard_01"}'

# -> 200 OK
{
  "token": "ey...",
  "status": "verified"
}
