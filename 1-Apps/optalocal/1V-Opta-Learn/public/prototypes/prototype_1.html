An operational, step-oriented guide focused on execution and verification within the Opta Local CLI environment.

1. Overview

The Opta CLI is a terminal-native command-line tool designed for precision engineering. It favors data density and raw output visibility over graphical fluff.

2. Configuration

Configure your local environment using the standard dotfiles. Ensure paths are absolute.
