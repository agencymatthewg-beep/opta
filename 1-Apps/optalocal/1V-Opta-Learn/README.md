# 1V-Opta-Learn

Canonical path: `1-Apps/optalocal/1V-Opta-Learn`

## Purpose

Guides, workflows, and learning hub for Opta Local as the first public Opta release, including canonical Opta AI activation patterns across local LMX and cloud runtimes.

## Local Development

```bash
cd <optalocal-root>/1V-Opta-Learn
npm install
npm run dev
```

## Validation

```bash
npm run check
```

## Build and Start

```bash
npm run build
npm run start
```

## Guide Tooling

```bash
npm run guides:inventory:check
npm run guides:validate
npm run guide:new
```
