# web

## Quick Context
- Opta application source code and documentation
- Contains: iOS, macOS, web, CLI implementations
- Use for: building and extending Opta products


Web applications for the Opta ecosystem built with React, Next.js, and Vite.

## Contents
- **optamize-website/** - Main marketing/documentation website (Cloudflare Workers)
- **opta-life/** - Life management web app (Next.js)
- **AICompare/** - AI model comparison tool
- **ai-components/** - Shared web components library

## Usage
Each sub-app has its own setup:
```bash
cd optamize-website
npm install && npm run dev

cd ../opta-life
npm install && npm run dev
```

Check individual README files in each app directory for specific setup instructions.
