# Opta Features

## Core Features

### Hardware Telemetry
Real-time monitoring of your system:
- **CPU**: Usage, temperature, core-by-core breakdown
- **GPU**: Usage, VRAM, temperature, clock speeds
- **RAM**: Used/available, swap usage
- **Disk**: Read/write activity

### Process Management
Take control of what's running:
- **Process List**: See all running processes with usage stats
- **Categorization**: System, Apps, Background, Gaming
- **Stealth Mode**: One-click kill of non-essential background processes
- **Safe List**: Protected system processes that won't be terminated

### Conflict Detection
Avoid optimization tool conflicts:
- Auto-detects: GeForce Experience, Razer Cortex, OMEN Hub, MSI Afterburner
- Warns about overlapping functionality
- Suggests which tools to disable

### Game Detection
Find and optimize your games:
- **Launchers**: Steam, Epic Games, GOG Galaxy
- **Auto-scan**: Finds installed games automatically
- **Manual add**: Add games Opta missed

## AI Features

### Local LLM
Privacy-first AI assistance:
- Runs on your machine (requires Ollama)
- No internet needed for basic queries
- Fast responses for routine questions

### Cloud AI
Advanced reasoning when needed:
- Complex optimization strategies
- Multi-game analysis
- Hardware upgrade recommendations
- Anonymized context (no personal data sent)

### Hybrid Routing
Best of both worlds:
- Simple questions - Local LLM (instant, private)
- Complex questions - Cloud AI (powerful, accurate)
- You control what goes where

## Optimization Engine

### Game Profiles
Per-game optimization:
- Community-verified settings
- Hardware-aware recommendations
- Before/after benchmarking

### Rollback
Safety first:
- Every change is reversible
- One-click undo
- Automatic backups before changes

### Transparency
Understand every change:
- Plain-language explanations
- Technical details available
- Impact predictions before applying

## Scoring System

### Opta Score
Your optimization metric:
- **Overall**: 0-100 composite score
- **Performance**: FPS, Stability, Load Times
- **Experience**: Quality, Thermal, Responsiveness
- **Competitive**: Input Lag, Latency, Interference

### Wow Factors
Shareable achievements:
- Money saved equivalent ("$200 GPU upgrade worth")
- Percentile ranking ("Top 15% of similar systems")
- Improvement timeline

### Leaderboards
Compare and compete:
- Similar hardware tier
- Price bracket
- Global ranking
- Friend comparisons

## Personalization

### User Profile
Opta learns your preferences:
- Tracks your optimization choices
- Identifies patterns over time
- Suggests based on history

### Recommendation Engine
Personalized suggestions:
- "Based on your history, prioritizing FPS"
- Dismissible recommendations
- Editable learned preferences

### Communication Style
Choose how Opta talks to you:
- **Educational**: Explains the "why"
- **Concise**: Just the facts
