# /gemini - Google Gemini

Use Google Gemini API for various tasks.

## Usage

```
/gemini [prompt]
```

## Capabilities

- General queries
- Image generation
- Video creation
- Code analysis
- Document analysis
- YouTube analysis

## MCP Tools

| Tool | Purpose |
|------|---------|
| `gemini-query` | General questions |
| `gemini-generate-image` | Create images |
| `gemini-analyze-image` | Analyze images |
| `gemini-generate-video` | Create videos |
| `gemini-search` | Grounded search |
| `gemini-analyze-document` | PDF/doc analysis |

## Output Format

```
**Gemini Response:**

[Response content]

[Image/video path if generated]
```

## Notes

- Supports pro and flash models
- Image generation with various aspect ratios
